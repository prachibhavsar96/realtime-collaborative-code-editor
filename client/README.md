# Realtime Code Editor Client

React, Vite, TypeScript, Socket.io-client, and Monaco Editor frontend for the collaborative code editor.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Vite runs on `http://localhost:5173` by default.

## Build

```bash
npm run build
```

## Environment

Create a `.env` file if you need to change the backend URL:

```bash
VITE_BACKEND_URL=http://localhost:4000
```
